# SkinCare-Hub
SkinCare Hub is a frontend-based skincare website built using HTML, CSS, and JavaScript, focused on delivering skincare information through a clean, modern, and professional user interface. The project emphasizes aesthetic design, intuitive navigation, and well-structured content to create an engaging and user-friendly experience.

The website showcases skincare routines, product-related content, and educational sections, with a strong focus on visual consistency, readability, and responsive layout. SkinCare Hub was developed as a personal project to combine web development skills with skincare awareness in a polished and professional format.
